## Features
- Finnish verb â†’ English meaning flashcards
- Reveal / Hide answer
- Next / Previous navigation
- Shuffle deck
- Progress counter (e.g. 3 / 20)
- Optional auto-reveal mode
- Keyboard shortcuts:
  - Right arrow: Next card
  - Left arrow: Previous card
  - R: Reveal
  - H: Hide

## How to use
1. Open the app in your browser.
2. Read the Finnish verb.
3. Click **Reveal** to check the English meaning.
4. Use **Next / Previous** to move through the deck.
